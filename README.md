# Jiu-Jitsu Championship Manager

## Programa para gerenciar um campeonato de Jiu-Jitsu

### Principais funções disponíveis:
- Cadastro competidor
- Remover competidor
- Alterar dados do competidor
- Pesquisar competidor
- Começar campeonato
